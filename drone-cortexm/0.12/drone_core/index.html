The core crate for Drone, an Embedded Operating System.

Documentation

  • Drone Book

Usage

Add the crate to your Cargo.toml dependencies:

[dependencies]
drone-core = { version = "0.12.0" }

Add or extend std feature as follows:

[features]
std = ["drone-core/std"]
